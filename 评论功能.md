## 在blog/views.py添加pub_comment
```
@require_POST
@login_required()
def pub_comment(request):
    blog_id = request.POST.get('blog_id')
    content = request.POST.get('content')
    BlogComment.objects.create(blog_id=blog_id,content=content,author=request.user)
    # 重新加载博客详情页
    return redirect(reverse('blog:blog_detail',kwargs={'blog_id':blog_id}))
```
## 在blog/urls.py添加
```
path('blog/comment',views.pub_comment,name='pub_comment'),
```

## 代码块高亮显示（highlight.js）
访问https://highlightjs.org/download下载
放进static文件夹中
在templates/blog_detail.html文件头部导入文件
```
{% block head %}
    <link rel="stylesheet" href="{% static 'highlightjs/styles/default.min.css' %}">
    <script src="{% static 'highlightjs/highlight.min.js' %}"></script>
{% endblock %}
```
尾部加入script标签
```
    <script>
        hljs.highlightAll();
    </script>
```
